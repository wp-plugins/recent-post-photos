== Description ==
Recent Photos Plugin provides with a widget to display n numbers of recent post photos from the media library in the sidebar.
If you find any error, please post them to me at asad@roohware.com

== Installation ==

= Requirements =
* PHP: 4.4.x or 5.x.x
* mySQL: 4.0, 4.1 or 5.x
* WordPress: 2.9 or newer

= Basic Installation =
* Plugin folder in the WordPress plugins folder must be `recent-photos`
* Upload folder `recent-post-photos` to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Go to WP Admin > Appereance > Widgets and put the 'Recent Post Photos' widget in your sidebar
* Configure the options as per your requirements and wordpress installation
